# Время компиляции контракта

## Оценка времени

### Нормальная компиляция
- **FunC → Fift**: 5-30 секунд
- **Fift → Cell**: 1-5 секунд
- **Итого**: **10-60 секунд** в большинстве случаев

### Если зависает дольше
- **1-2 минуты** - еще нормально для сложных контрактов
- **3-5 минут** - может быть проблема, но стоит подождать
- **> 5 минут** - вероятно ошибка, нужно проверить логи

## Факторы, влияющие на время

1. **Размер контракта**: Наш контракт ~290 строк - средний размер
2. **Сложность**: Использование словарей может замедлить компиляцию
3. **Производительность системы**: На старых компьютерах может быть медленнее
4. **Первый запуск**: Может быть медленнее из-за загрузки библиотек

## Что делать, если зависает

### Если компиляция занимает > 2 минут:

1. **Проверьте процесс**:
   ```bash
   ps aux | grep func
   ```

2. **Попробуйте прервать** (Ctrl+C) и запустить снова

3. **Проверьте ошибки**:
   ```bash
   /usr/local/Cellar/ton/64/bin/func sources/MarketEscrow.fc -o build/MarketEscrow.fif 2>&1 | tee compile.log
   ```

4. **Попробуйте компилировать по частям** - проверьте синтаксис сначала

## Типичные проблемы

### Ошибка компиляции
- Проверьте синтаксис FunC
- Убедитесь, что stdlib.fc доступен
- Проверьте пути к файлам

### Зависание
- Может быть проблема с памятью
- Попробуйте перезапустить терминал
- Проверьте, не запущено ли несколько процессов func

## Рекомендации

1. **Запускайте компиляцию в отдельном терминале** - так вы сможете видеть прогресс
2. **Используйте `time` для измерения**:
   ```bash
   time ./scripts/compile.sh
   ```
3. **Если долго** - подождите до 2-3 минут перед прерыванием

## Ожидаемое время для нашего контракта

**Реалистичная оценка**: **15-45 секунд**

Контракт среднего размера (~290 строк), использует словари, но не очень сложный.

---

**Вывод**: Если компиляция занимает больше 2-3 минут, вероятно есть проблема. В норме должно быть 10-60 секунд.
